# Provenance.io WalletConnect-JS QRCode Modal

Module to display QRCode Modal, linking Web UI with Provenance WalletConnect-JS.

[Provenance] is a distributed, proof of stake blockchain designed for the financial services industry.

For more information about [Provenance Inc](https://provenance.io) visit https://provenance.io
